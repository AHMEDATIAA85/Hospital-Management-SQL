Hospital Management System (SQL)
A comprehensive database solution for managing hospital operations, financial reporting, and staff performance.

Project Overview
This project demonstrates a robust SQL Server database designed to handle daily hospital activities. It manages relationships between doctors, medical services, and patients operations while providing deep insights into revenue and department performance.

Technical Implementation
Relational Data Modeling: Created a structured schema using Primary and Foreign Keys to ensure data integrity.

Financial Analytics: Developed queries to calculate total income, doctor salaries, and service frequency using SUM, COUNT, and AVG.

Advanced Logic:

Implemented Subqueries to identify top and lowest-performing doctors.

Used HAVING clauses for post-aggregation filtering.

Applied LIKE and BETWEEN operators for precise data searching and range filtering.

Data Maintenance: Efficiently managed data through structured UPDATE and DELETE operations.
